# Stock Chart Dashboard

This is a stock analysis dashboard built with Python and Streamlit.

## Features
- Candlestick chart visualization
- SMA20 / SMA50 indicators
- RSI indicator
- Buy / Sell signal hints
- Interactive stock symbol input
- Technical analysis summary

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly
- yfinance
